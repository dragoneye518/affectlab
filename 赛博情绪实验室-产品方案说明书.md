# 赛博情绪实验室 - 产品方案说明书

## 1. 产品概述
**赛博情绪实验室 (Cyber Emotion Lab)** 是一款面向 Gen Z 群体的像素风情绪生成器小程序。用户通过选择特定的“情绪模版”并输入当前的想法或状态，利用 AI（DeepSeek）将枯燥的文字转化为具有“互联网嘴替”风格的赛博文案，并生成一张高颜值的像素风卡片（分为 N/R/SR/SSR 稀有度）。

核心价值：**情绪嘴替 + 社交货币 + 盲盒体验**。

## 2. 核心功能模块

### 2.1 首页 (Home)
- **Season 1 Finale 活动**：顶部展示 "2026 赛博庙会" 倒计时 Banner，营造紧迫感与活动氛围。
- **模版列表**：展示所有可用模版，采用 2 列瀑布流/网格布局。
  - **模版卡片**：展示封面图、标题、标签（HOT/NEW/LIMITED）、消耗糖果数。
  - **7大核心模版**：
    1. **今日能量·指南 (Energy Daily)**: 运势与行动建议。
    2. **职场·嘴替 (Roast Boss)**: 吐槽工作与老板。
    3. **前任·回复 (Ex Reply)**: 霸气或释怀的回应。
    4. **修勾·传声筒 (Pet Voice)**: 宠物的内心戏。
    5. **失眠·壁纸 (Sleep Wallpaper)**: 深夜emo与助眠。
    6. **亲戚·护盾 (Relative Shield)**: 应对春节亲戚盘问。
    7. **奶茶·借口 (Diet Excuse)**: 减肥路上的绊脚石。
  - **分类导航**：
    - **胶囊 (Home)**：主列表。
    - **雷达 (Radar)**：分类筛选（能量指南、高情商嘴替、社交人设、2026未来）及热门标签搜索。
    - **背包 (Mine)**：查看历史记录。
- **状态栏**：
  - 显示当前糖果数量（Candy）。
  - **每日补给**：每日签到领取 +10 糖果。
  - **看广告**：观看模拟广告领取 +5 糖果。
- **自定义入口**：底部列表末尾提供“信号转译”通用入口，支持任意主题输入，不受模版限制。

### 2.2 情绪输入 (Input)
- **模版详情**：展示当前选中的模版信息及消耗。
- **AI 信号转译 (DeepSeek)**：
  - 用户输入原始文本（如“不想上班”）。
  - 点击“AI 信号转译”，调用 DeepSeek API 生成三种风格的文案：
    - **TOXIC (毒舌/幽默)**: 犀利、搞笑、攻击性强。
    - **EMO (氛围/深沉)**: 伤感、诗意、孤独。
    - **GLITCH (故障/机械)**: 抽象、机器人、系统错误风格。
  - 用户选择一种风格填入输入框。
- **生成确认**：扣除糖果，开始生成。

### 2.3 生成结果 (Result)
- **生成动画**：展示 "INITIALIZING NEURAL LINK..." 等赛博风格日志与进度条，配合 Canvas 粒子特效 (Confetti)。
- **结果卡片 (HoloCard)**：
  - **3D 悬浮效果**：支持触摸跟随的 3D 倾斜特效 (Tilt Effect)。
  - **稀有度系统**：基于概率随机生成，不同稀有度对应不同边框与特效：
    - **N (Common)**: 基础款。
    - **R (Rare)**: 稀有款。
    - **SR (Super Rare)**: 超稀有。
    - **SSR (Ultra Rare)**: 传说款，拥有专属流光特效与高分评价。
  - **内容展示**：合成的像素图 + AI 润色后的文案 + 幸运评分 (0-100)。
- **操作选项**：
  - **保存/分享**：保存图片或分享给好友。
  - **再来一次**：重新生成（如糖果不足提示看广告）。

### 2.4 我的背包 (Mine)
- **历史记录**：展示用户生成过的所有卡片。
- **筛选/查看**：点击查看大图，回顾过往情绪。

## 3. 用户体验细节 (UX)
- **视觉风格**：高对比度霓虹色（#ff00ff, #00ffff, #00ff00）+ 纯黑背景 (#050505)，像素艺术风格 (Pixel Art)。
- **交互反馈**：
  - 按钮点击缩放 (Active Scale)。
  - 霓虹辉光阴影与故障风 (Glitch) 动画。
  - 完善的 Toast 提示和加载动画。
- **防呆设计**：
  - 糖果不足时引导看广告。
  - API 失败时自动降级为本地 Fallback 文案，确保流程不中断。

## 4. 商业化/积分体系
- **糖果 (Candy)**：核心消耗代币。
  - 初始赠送：20 颗。
  - 每次生成消耗：1-3 颗（视模版而定）。
  - 获取途径：
    - 每日签到 (+10)。
    - 观看广告 (+5)。
